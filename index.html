FlowArray

The FlowArray class is a simple class designed to work with multiple classes of data at a time and to allow dinamicity with class types used.

In order to use this class just simply import it using your IDE (you may have to look up how to do it if you do not know how to already). Click here and then change "<IDE>" to the name of your IDE.
